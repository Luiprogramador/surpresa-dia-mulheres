# Surpresa - Dia das Mulheres 🌷

Um projeto especial e interativo desenvolvido para criar uma experiência única e homenagear uma pessoa muito importante. 

## 💻 Sobre o Projeto

Este site foi construído inteiramente com tecnologias web clássicas (HTML, CSS e JavaScript) e pensado para ser uma surpresa digital. Ele conta com uma interface de design divertido, mensagens especiais e uma galeria de fotos integrada para relembrar bons momentos de forma dinâmica.

## ✨ Funcionalidades

- **Design Divertido e Responsivo:** Uma interface estilizada, agradável e feita com muito carinho.
- **Navegação por Paginação:** Transição fluida e organizada entre as mensagens e seções do site.
- **Slideshow de Fotos Imersivo:** Um carrossel de imagens com efeitos e transições para exibir fotos especiais com o encaixe perfeito.
- **Efeitos Visuais:** Animações interativas para tornar a experiência de quem está navegando ainda mais legal.

## 🛠️ Tecnologias Utilizadas

- **HTML5:** Estruturação semântica de todo o conteúdo.
- **CSS3:** Estilização, cores, layouts e animações visuais.
- **JavaScript:** Lógica de controle da paginação e funcionamento do slideshow/carrossel de fotos.

## 📁 Estrutura do Projeto

* `assets/`: Diretório destinado ao armazenamento das imagens e mídias do projeto.
* `css/`: Arquivos de estilo responsáveis por todo o visual da página.
* `js/`: Scripts com a lógica de interatividade.
* `index.html`: Arquivo principal e porta de entrada do site.

## 🚀 Como visualizar

Como é um projeto puramente Front-end estático, rodar é muito simples:

1. Clone este repositório na sua máquina:
   git clone [https://github.com/Luiprogramador/surpresa-dia-mulheres.git](https://github.com/Luiprogramador/surpresa-dia-mulheres.git)
   
2. Navegue até a pasta do projeto.

3. Dê um duplo clique no arquivo index.html para abri-lo diretamente no seu navegador de preferência.
